Links

This example is about links and images

there are 3 types of links which exist.

these being:

  1. Absolute links
    these are the most detailed and refer to the complete web address example: "https://developer.mozilla.org/en-US/docs/Web/HTML"
    Mozilla Developer Network
  2. Relative links
    these point to another page on your website view from the current page
    we can link to the extra file by supplying the path like this : "misc/extras.html"
    extras
  3. Root-relative links
    These are similar to relative links but they are relative to the root from the entire website.
    since we are using local html files, we won't be able to expiriment with root-relative links.

    4. the only difference between a root-relative link and a relative links is: one starts with a forward slash, this represent the root of your site.

targets:
we can use the target tag in a link to specifiy where we want the new web page to open. if you don't add it, it'll open on the current page by default
adding a "target = _blank" will make sure the webpage opens on a new tab.
